# Audio Files for Menu

## Required File

Place your menu music file here with the name:
- **menu.mp3**

## Supported Formats
- MP3 (.mp3)
- M4A (.m4a)
- OGG (.ogg)
- WAV (.wav)

## Notes
- The audio will be sent as a regular audio message (not voice note)
- Make sure the file size is reasonable for WhatsApp (max 16MB recommended)
- The audio will play after the menu image is sent

## How to Add Your Music
1. Find or create your menu music
2. Convert it to MP3 format (recommended)
3. Rename it to `menu.mp3`
4. Place it in this folder: `WhatsApp/database/audio/menu.mp3`
5. Restart your bot

That's it! Now all menu commands will send the image + music automatically.
